Modutwin

Modular Indoor Digital Twins via Door-Based Alignment of 3D Gaussian Splatting Assets

Sejong University · AllIsWell
Paper Code Demo Video Website

Room-level 3DGS assets are refined and connected through the rectangular structure of doors.

Abstract

This work proposes a modular indoor digital twin construction method based on room-level 3D Gaussian Splatting assets. To prevent visual conflicts between adjacent modules, Gaussians outside the room boundary or crossing the boundary are removed, and their visual information is baked into boundary mesh textures. New room modules are then aligned to the existing digital twin using the four corners of corresponding door regions.

Method Overview

Users upload room-level 3DGS assets, define room boundary planes, and specify door regions using four corners. Door corners can be selected manually in the viewer or estimated with SAM3. The system refines each asset, separates the door region, and computes the transform needed to connect the new module to the existing scene.
